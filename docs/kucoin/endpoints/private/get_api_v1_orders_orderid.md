# GET /api/v1/orders/{orderId}

**Source:**
[/api/v1/orders/{orderId}](https://www.kucoin.com/docs/rest//api/v1/orders/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Get Order By OrderId - Old

Request a single order info by order ID via this endpoint.

## Parameters

| Parameter | Required | Type   | Description                                         |
| --------- | -------- | ------ | --------------------------------------------------- |
| orderId   | required | string | The unique order id generated by the trading system |
