# DELETE /api/v1/stop-order/{orderId}

**Source:** [/api/v1/stop-order/{orderId}](https://www.kucoin.com/docs/rest//api/v1/stop-order/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Cancel Stop Order By OrderId

This endpoint can be used to cancel a spot stop order by orderId. 

## Parameters

| Parameter | Required | Type | Description |
|-----------|----------|------|-------------|
| orderId | required | string | The unique order id generated by the trading system |

## Responses

### 200

| Parameter | Required | Type | Description |
|-----------|----------|------|-------------|
| code | required | string |  |
| data | required | object |  |
| data.cancelledOrderIds | required | array | order ID array |
