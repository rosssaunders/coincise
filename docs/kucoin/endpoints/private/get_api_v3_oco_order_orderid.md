# GET /api/v3/oco/order/{orderId}

**Source:** [/api/v3/oco/order/{orderId}](https://www.kucoin.com/docs/rest//api/v3/oco/order/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Get OCO Order By OrderId

Request via this interface an OCO order information via the order ID.

## Parameters

| Parameter | Required | Type | Description |
|-----------|----------|------|-------------|
| orderId | required | string | The unique order id generated by the trading system |

## Responses

### 200

| Parameter | Required | Type | Description |
|-----------|----------|------|-------------|
| code | required | string |  |
| data | required | object |  |
