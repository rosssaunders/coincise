# GET /api/v3/oco/order/{orderId}

**Source:**
[/api/v3/oco/order/{orderId}](https://www.kucoin.com/docs/rest//api/v3/oco/order/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Get OCO Order By OrderId

Request via this interface an OCO order information via the order ID.

## Parameters

| Parameter | Required | Type   | Description                                         |
| --------- | -------- | ------ | --------------------------------------------------- |
| orderId   | required | string | The unique order id generated by the trading system |

## Responses

### 200

| Parameter      | Required | Type    | Description                                                                                                                   |
| -------------- | -------- | ------- | ----------------------------------------------------------------------------------------------------------------------------- |
| code           | required | string  |                                                                                                                               |
| data           | required | object  |                                                                                                                               |
| data.symbol    | required | string  | symbol                                                                                                                        |
| data.clientOid | required | string  | Client Order ID                                                                                                               |
| data.orderId   | required | string  | The unique order ID generated by the trading system, which can be used later for further actions such as canceling the order. |
| data.orderTime | required | integer | Order placement time, milliseconds                                                                                            |
| data.status    | required | string  | Order status: NEW: New, DONE: Completed, TRIGGERED: Triggered, CANCELED: Canceled                                             |
