# POST /api/v1/position/margin/deposit-margin

**Source:** [/api/v1/position/margin/deposit-margin](https://www.kucoin.com/docs/rest//api/v1/position/margin/deposit-margin)

## Authentication

Required (Private Endpoint)

## Description

Add Isolated Margin

Add Isolated Margin Manually.

## Request Body

| Parameter | Required | Type | Description |
|-----------|----------|------|-------------|
| symbol | required | string | Symbol of the contract, Please refer to [Get Symbol endpoint: symbol](https://www.kucoin.com/docs-new/api-3470220)  |
| margin | required | number | Margin amount (min. margin amount≥0.00001667XBT） |
| bizNo | required | string | A unique ID generated by the user, to ensure the operation is processed by the system only once, The maximum length cannot exceed 36 |

|-----------|----------|------|-------------|
