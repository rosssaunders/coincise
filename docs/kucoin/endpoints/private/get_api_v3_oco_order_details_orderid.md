# GET /api/v3/oco/order/details/{orderId}

**Source:**
[/api/v3/oco/order/details/{orderId}](https://www.kucoin.com/docs/rest//api/v3/oco/order/details/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Get OCO Order Detail By OrderId

Request via this interface to get a oco order detail via the order ID.

## Parameters

| Parameter | Required | Type   | Description                                         |
| --------- | -------- | ------ | --------------------------------------------------- |
| orderId   | required | string | The unique order id generated by the trading system |

## Responses

### 200

| Parameter               | Required | Type    | Description                                                                                                                  |
| ----------------------- | -------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| code                    | required | string  |                                                                                                                              |
| data                    | required | object  |                                                                                                                              |
| data.orderId            | required | string  | The unique order id generated by the trading system,which can be used later for further actions such as canceling the order. |
| data.symbol             | required | string  | symbol                                                                                                                       |
| data.clientOid          | required | string  | Client Order Id                                                                                                              |
| data.orderTime          | required | integer | Order placement time, milliseconds                                                                                           |
| data.status             | required | string  | Order status: NEW: New, DONE: Completed, TRIGGERED: Triggered, CANCELLED: Cancelled                                          |
| data.orders             | required | array   |                                                                                                                              |
| data.orders[].id        | required | string  |                                                                                                                              |
| data.orders[].symbol    | required | string  |                                                                                                                              |
| data.orders[].side      | required | string  |                                                                                                                              |
| data.orders[].price     | required | string  |                                                                                                                              |
| data.orders[].stopPrice | required | string  |                                                                                                                              |
| data.orders[].size      | required | string  |                                                                                                                              |
| data.orders[].status    | required | string  |                                                                                                                              |
