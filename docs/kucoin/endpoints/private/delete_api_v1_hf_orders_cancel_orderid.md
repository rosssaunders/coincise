# DELETE /api/v1/hf/orders/cancel/{orderId}

**Source:**
[/api/v1/hf/orders/cancel/{orderId}](https://www.kucoin.com/docs/rest//api/v1/hf/orders/cancel/{orderId})

## Authentication

Required (Private Endpoint)

## Description

Cancel Partial Order

This interface can cancel the specified quantity of the order according to the
orderId. The order execution order is: price first, time first, this interface
will not change the queue order

## Parameters

| Parameter  | Required | Type   | Description                                         |
| ---------- | -------- | ------ | --------------------------------------------------- |
| orderId    | required | string | The unique order id generated by the trading system |
| symbol     | required | string | symbol                                              |
| cancelSize | required | string | The size you want cancel                            |

## Responses

### 200

| Parameter       | Required | Type   | Description           |
| --------------- | -------- | ------ | --------------------- |
| code            | required | string |                       |
| data            | required | object |                       |
| data.orderId    | required | string | order id              |
| data.cancelSize | required | string | The size you canceled |
