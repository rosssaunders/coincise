# GET /api/v1/fee-rate-tiers

## List fee rate tiers

Return all the fee rate tiers.

**Operation ID:** getFeeRateTiers

**Tags:** Fee Rates

**Endpoint:** `GET /api/v1/fee-rate-tiers`

**Base URL:** https://api.international.coinbase.com

## Authentication

This endpoint does not require authentication.

## Responses

### 200

Fee rate tiers list
