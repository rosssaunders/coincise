# GET /api/v1/position-offsets

## List position offsets

Returns all active position offsets

**Operation ID:** getPositionOffsets

**Tags:** Position Offsets

**Endpoint:** `GET /api/v1/position-offsets`

**Base URL:** https://api.international.coinbase.com

## Authentication

This endpoint does not require authentication.

## Responses

### 200

Position offsets list
