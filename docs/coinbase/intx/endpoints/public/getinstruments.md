# GET /api/v1/instruments

## List instruments

Returns all of the instruments available for trading.

**Operation ID:** getInstruments

**Tags:** Instruments

**Endpoint:** `GET /api/v1/instruments`

**Base URL:** https://api.international.coinbase.com

## Authentication

This endpoint does not require authentication.

## Responses

### 200

Instrument list
