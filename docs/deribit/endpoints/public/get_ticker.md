# GET /public/ticker

Get ticker for an instrument.

### Parameters

| Parameter       | Required | Type   | Enum            | Description |
| --------------- | -------- | ------ | --------------- | ----------- |
| instrument_name | true     | string | Instrument name |             |

### Response

| Name                            | Type    | Description                                                                                                                                                                                                            |
| ------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id                              | integer | The id that was sent in the request                                                                                                                                                                                    |
| jsonrpc                         | string  | The JSON-RPC version (2.0) result object                                                                                                                                                                               |
| result.ask_iv                   | number  | (Only for option) implied volatility for best ask                                                                                                                                                                      |
| result.best_ask_amount          | number  | It represents the requested order size of all best asks                                                                                                                                                                |
| result.best_ask_price           | number  | The current best ask price, null if there aren't any asks                                                                                                                                                              |
| result.best_bid_amount          | number  | It represents the requested order size of all best bids                                                                                                                                                                |
| result.best_bid_price           | number  | The current best bid price, null if there aren't any bids                                                                                                                                                              |
| result.bid_iv                   | number  | (Only for option) implied volatility for best bid                                                                                                                                                                      |
| result.current_funding          | number  | Current funding (perpetual only)                                                                                                                                                                                       |
| result.delivery_price           | number  | The settlement price for the instrument. Only when state = closed                                                                                                                                                      |
| result.estimated_delivery_price | number  | Estimated delivery price for the market. For more details, see Contract Specification > General Documentation > Expiration Price                                                                                       |
| result.funding_8h               | number  | Funding 8h (perpetual only)                                                                                                                                                                                            |
| result.greeks                   | object  | Only for options                                                                                                                                                                                                       |
| result.greeks.delta             | number  | (Only for option) The delta value for the option                                                                                                                                                                       |
| result.greeks.gamma             | number  | (Only for option) The gamma value for the option                                                                                                                                                                       |
| result.greeks.rho               | number  | (Only for option) The rho value for the option                                                                                                                                                                         |
| result.greeks.theta             | number  | (Only for option) The theta value for the option                                                                                                                                                                       |
| result.greeks.vega              | number  | (Only for option) The vega value for the option                                                                                                                                                                        |
| result.index_price              | number  | Current index price                                                                                                                                                                                                    |
| result.instrument_name          | string  | Unique instrument identifier                                                                                                                                                                                           |
| result.interest_rate            | number  | Interest rate used in implied volatility calculations (options only)                                                                                                                                                   |
| result.interest_value           | number  | Value used to calculate realized_funding in positions (perpetual only)                                                                                                                                                 |
| result.last_price               | number  | The price for the last trade                                                                                                                                                                                           |
| result.mark_iv                  | number  | (Only for option) implied volatility for mark price                                                                                                                                                                    |
| result.mark_price               | number  | The mark price for the instrument                                                                                                                                                                                      |

| result.open_interest            | number  | The total amount of outstanding contracts in the corresponding amount units. For perpetual and inverse futures the amount is in USD units. For options and linear futures and it is the underlying base currency coin. |
| result.settlement_price         | number  | Optional (not added for spot). The settlement price for the instrument. Only when state = open                                                                                                                         |

| result.stats.low                | number  | Lowest price during 24h                                                                                                                                                                                                |
| result.stats.price_change       | number  | 24-hour price change expressed as a percentage, null if there weren't any trades                                                                                                                                       |

| result.timestamp                | integer | The timestamp (milliseconds since the Unix epoch)                                                                                                                                                                      |
| result.underlying_index         | number  | Name of the underlying future, or index_price (options only)                                                                                                                                                           |
| result.underlying_price         | number  | Underlying price for implied volatility calculations (options only)                                                                                                                                                    |
