# GET Subscribe to unified account asset push

**Source:**
[Subscribe to unified account asset push](https://www.htx.com/en-us/opend/newApiPages/?id=10000084-77b7-11ed-9966-0242ac110003)

**Category:** USDT-M Unified Account

## Authentication

Required (Private Endpoint)

### accounts_unify.USDT (Subscribe to unified account asset push)

Signature verification: Yes

Interface permission: Read

Interface description: It is used for real-time monitoring of asset changes in
the unified account.
