## Unsubscribe

The format of unsubscription is as follows:

{ "id": "id1", "reqType": "unsub", "dataType": "data to unsub"}

Confirmation of Unsubscription:

{ "id": "id1", "code": 0, "msg": "" }

### Symbol Description

Symbol must be fully capitalized
