# Error Codes

## Error Codes

-   Any endpoint can return an ERROR

Sample Payload below:

```
{  "code": -1121,  "msg": "Invalid symbol."}
```

-   Specific error codes and messages are defined in [Errors Codes](/docs/binance-spot-api-docs/errors).

> Source: [https://developers.binance.com/docs/binance-spot-api-docs/rest-api/error-codes](https://developers.binance.com/docs/binance-spot-api-docs/rest-api/error-codes)
